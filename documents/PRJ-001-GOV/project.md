## Overview

Stand up PMO as Code's own governance: a documents repository covering the
whole portfolio, gated the same way we ask adopters to gate theirs, with a
public derived dashboard and, once the engine supports per-project repo
mapping, the execution bridge running against each component's code repo.
This project ends when the portfolio governs itself with no manual steps.

## Scope

This repository, its gates and dashboard, and bridge wiring to each
component repository. Product content of the components is out of scope.
